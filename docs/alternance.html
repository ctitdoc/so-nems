Rust-Wasm learning curve

Rust : a general purpose "new generation" strongly typed language :

  • with strong and stable community,

  • from Stackoverflow survey 2021 : "For the sixth-year, Rust is the most loved language".

Wasm (WebAssembly) : a standard, safe and portable bytecode format, executable in browsers, with or without Javascript code.

The basic principle of a Wasm Web App :

  • code your whole web app in your desired "wasm compliant" language => JS is no more mandatory browser side !

  • compile your browser side code in Wasm;

  • optionally use valued-added JS libraries => JS and Wasm are totally interoperable !

Rust key concepts :

  • Type system : Struct, Trait, Impl, Fn, Enum, Generic types, type inference => flexible ans stable object oriented programming (yet more complex to learn),

  • Borrowing : value vs references managment => no garbage collecting needed = less resource consumption (yet more complex to learn),

  • Macros : add user defined languages => see html language macro below,

  • Async : asynchronous programming => required in web browser programming.
